# Registration Form Project

## Description
A simple and responsive Registration Form created using HTML and CSS. The project includes personal details, course selection, delivery mode, and form validation elements.

---

## Features
- Personal details section
- Course registration section
- Gender selection
- Country dropdown
- File upload option
- Responsive design

---

## Technologies Used
- HTML
- CSS

---

## Project Structure
registration-form/
│── index.html
│── style.css
│── README.md

---

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in browser
